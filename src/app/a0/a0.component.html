A0: Hello, World!

Typically, the first piece of code that a person writes in a programming language is the Hello, World! program. This assignment will have you do just that in HTML, getting you used to editing files. As we will discuss in the coming weeks, HTML is an odd language for writing a Hello World! program, since many plain text files can be valid HTML. But since it's the language that we'll spend the most time with, it's a valuable to start here. As an A0, this assignment isn't intended to be challenging. It's primarily intended to make sure your environment is set up for more complex assignments later.

Starter Code

The code is available for download on GitHub. If you want to practice version control, you can use create your own copy of the repository via "use this template" in the top-right. You can also directly clone/download the repository via the "code" button.

This assignment contains one single file that you will edit: index.html.

Editing and Running your Code

We recommend Visual Studio Code as an Integrated Development Environment (IDE). But you can use whatever program you want to edit your code. To run your site, you can open it up in your favorite browser by double-clicking on it or navigating in your browser to open the file.
